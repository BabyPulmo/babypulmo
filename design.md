## 6. Non-negotiable for any UI proposal

1. **Bangla strings stay verbatim** from `babypulmo/lib/tts.ts::STOCK_BANGLA` (loaded from the private `BabyPulmo/clinical-content` repo at runtime). Never generate Bangla in components.
2. **No runtime LLM call in any UI** — all clinical decisions hit the API → `lib/claude.ts::decideSeverityMultiModal` which is deterministic. UI never asks an LLM "what's wrong with this child."
3. **§3 ARCHITECTURE.md carve-out** governs where LLMs are/aren't allowed. UI must not contradict it.
4. **Accessibility**: all interactive elements have `aria-label`, keyboard focus rings, ≥4.5:1 contrast on text.
